Project: Food Connect (Web Development)

Role: Lead Developer

Project Overview

Food Connect is an intergenerational web application designed to foster community engagement by allowing youth and seniors to share both traditional and modern recipes. The goal was to bridge the intergenerational divide while preserving cultural culinary heritage.

Project Visuals & Demo

Event Landing Page

Event Details

User Authentication: Demonstrating the secure sign-in process for user profiles.

Event Interaction: Users can view details, join, or unjoin community cooking events in real-time.

Work Process

  • Planning: Conducted user journey mapping to identify accessibility hurdles for seniors, resulting in a high-contrast UI and simplified navigation.
  • Development: Engineered the backend using Python/Flask, implementing Jinja2 logic to dynamically serve recipe content based on user categories.
  • Data Security: Developed a relational database schema in SQLite3 with password hashing to ensure secure and efficient member data storage.

Skills Applied

Python, Flask, Jinja2, sqlite3, HTML, CSS, JavaScript.

Results

Delivered a fully functional intergenerational platform that bridges the digital gap. The final application successfully streamlined recipe sharing and community event management, ensuring accessibility for users aged 13 to 70+.
